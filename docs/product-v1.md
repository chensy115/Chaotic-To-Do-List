# 产品 V1 · 赛博抬杠待办（当前基线）

> **文档状态**：已发布（当前线上 / 本地运行的产品版本）  
> **最后更新**：2026-07-03  
> **说明**：V1 指**整款产品**的能力边界，不是某次 UI 改版。界面专项见 [ui-redesign.md](./ui-redesign.md)。

---

## 1. 产品定位

**赛博抬杠待办** — 劝退型 Todo：加任务被 AI（或本地引擎）怼，完成按钮会逃跑，甩锅有借口，画饼有指数。

气质：**抬杠、混乱、反内卷**。不是生产力工具，是带游戏感的单机 Web 玩具。

---

## 2. V1 能力范围

### 2.1 核心玩法

| 能力 | 说明 |
|------|------|
| 抬杠加任务 | 输入任务 → AI/本地劝退 →「好吧算了」或「我偏要加」（需怼两次） |
| 待办战场 | 任务以关卡卡片展示，含腐败度、抓捕区 |
| 逃跑打卡 | 「✓ 完成打卡」在抓捕区内随机逃跑，抓住才算完成 |
| 明日再战 | 甩锅 + 随机借口，影响腐败与画饼指数 |
| 画饼指数 | 底部 HUD：劝退 / 逃跑 / 甩锅 / 完成统计 + 摆烂称号 |

### 2.2 AI 与本地

| 能力 | 说明 |
|------|------|
| 本地抬杠引擎 | 关键词、时段、通用台词，`roastEngine.ts` |
| 真实 AI | OpenAI 兼容 API，流式输出，Vite 本地代理 |
| 配置 | 界面配置 API Key，或 `.env`；Key 存 localStorage |

### 2.3 体验与呈现

| 能力 | 说明 |
|------|------|
| 聊天室 + HUD 布局 | 左对话 / 右待办 / 底统计（详见 [ui-redesign.md](./ui-redesign.md)） |
| Mascot | SVG 吉祥物，随 loading / 抬杠 / 倔强切换 mood |
| 主题 | 霓虹 / 深空两套 `data-theme` |
| 新手引导 | 三步聚光灯，localStorage 记一次 |
| 动效 | confetti、数字滚动、腐败 glitch 等 |

### 2.4 数据

| 能力 | 说明 |
|------|------|
| 持久化 | 待办 `chaotic-todos`、统计 `chaotic-stats`、主题、API 配置 |
| 范围 | 纯浏览器 localStorage，无账号、无云同步 |
