# 备份与迁移

> 无账号、无后端，用 JSON 文件备份「摆烂档案」。

## 怎么用

1. 展开底部 **HUD 档案**
2. 点击 **导出文件** — 下载 `chaotic-backup-YYYYMMDD.json`
3. 存到网盘、发给自己或发给朋友
4. 换设备或重装后，在 **导入** 处选择该文件即可恢复

## 备份里有什么

`chaotic-backup-YYYYMMDD.json` 结构（`STORAGE_VERSION: 3`）：

| 字段 | 内容 |
|------|------|
| `tasks` | 全部待办与已完成 |
| `stats` | 劝退 / 逃跑 / 甩锅 / 完成 / 认命次数 |
| `badges` | 已解锁徽章 |
| `timeline` | 摆烂日志（最近 500 条） |
| `snapshots` | 近 14 日画饼快照 |

**不包含**：AI API Key（Key 单独存在 `chaotic-api-config`）。

## 安全提示

- 备份含你的任务文案与 AI 抬杠记录，分享前请自行脱敏
- 不要使用公开链接分享含隐私内容的备份文件

## 相关入口

- **导出文件** — 下载 JSON 到本地
- **导入** — 恢复备份（见应用内校验提示）
